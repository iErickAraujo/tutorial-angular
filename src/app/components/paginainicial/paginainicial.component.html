Tutorial de instalação do Angular

Node

Começaremos baixando o node no computador, recomendamos que baixe a versão LTS. Para baixar o node clique aqui.

LTS significa Suporte de Longo Prazo e a versão recomendada para a maioria dos usuários. Nodejs org. faz novas versões frequentemente com novos recursos, correções de bugs e otimizações de desempenho. Como regra geral, podemos assumir com segurança que a versão mais recente é sempre a melhor versão a ser usada.

Node Version Manager

Em seguida instalaremos o nvm, para um melhor gerenciamento das versões do node clique aqui.

O Node Version Manager, mais comumente chamado de NVM, é a maneira mais popular de instalar várias versões do Node.js. Mas só está disponível para Mac/Linux e não tem suporte no Windows. Em vez disso, recomendamos instalar o nvm-Windows e usá-lo para instalar o Node.js e o npm (Gerenciador de Pacotes do Node).

Criando uma pasta para armazenar os projetos

Para facilitar a instalação e prevenir erros, recomendamos que crie sua pasta dentro da pasta Documentos.

Angular

Para iniciarmos a instalação do Angular abra o prompt de comando no modo administrador caso tenha acesso, caso não tenha clique aqui. No prompt digite o seguinte comando npm install -g @angular/cli@13.0.0 ou npm i -g @angular/cli@13.0.0 e pressione Enter, depois de instalado o próximo passo e abrir a pasta no prompt de (usaremos a pasta com o nome TutorialAngular para fins de explicação).

Usando o comando cd e cd.. . O comando cd é usado para entrar em uma pasta apertando espaço e digitando o nome da pasta.
    Exemplo:
  • C:\Users\Usuario cd Documentos
  • Aperte Enter
  • C:\Users\Usuario\Documentos
Já o comando cd.. e utilizado para sair de uma pasta.
    Exemplo:
  • C:\Users\Usuario\Documentos cd..
  • Aperte Enter
  • C:\Users\Usuario

No prompt de volta a pasta Documentos entraremos na pasta que criamos anteriormente usando o comando cd.

  • C:\Users\Usuario\Documentos cd TutorialAngular

Dentro da pasta TutorialAngular digite o comando a seguir para criar o projeto (Novamente usaremos o nome TutorialAngular e recomendamos que use o mesmo nome de sua pasta).

  • C:\Users\Usuario\Documentos\TutorialAngular ng new TutorialAngular

Para criar juntamente o arquivo de rota aperte y e Enter.

Escolha o formato de folha de estilo que você irá usar. Apertando as setas do teclado para Cima e para Baixo e então pressione Enter.

O próximo passo entre dentro da pasta do projeto usando cd 'nome do projeto'.

  • C:\Users\Usuario\Documentos\TutorialAngular cd TutorialAngular

Componentes

Ainda na pasta do projeto digite.

  • C:\Users\Usuario\Documentos\TutorialAngular\TutorialAngular ng generate components 'nome do componente'

    • ou

  • C:\Users\Usuario\Documentos\TutorialAngular\TutorialAngular ng g c 'nome do componente'

Pressione Enter este código cria arquivos component.

Módulos

Dentro da pasta do projeto escreva o seguinte.

  • C:\Users\Usuario\Documentos\TutorialAngular\TutorialAngular ng generate modules 'nome do modulo'

    • ou

  • C:\Users\Usuario\Documentos\TutorialAngular\TutorialAngular ng g m 'nome do modulo'

Pressione Enter este código cria um arquivo module.

Abrindo o projeto

Para iniciar o projeto digite.

  • C:\Users\Usuario\Documentos\TutorialAngular\TutorialAngular ng serve -o

Aguarde enquanto o projeto abre o navegador padrão. Feito isso minimize o prompt e você está pronto(a) para desenvolver usando uma IDE de sua escolha.

Muito cuidado a pagina só pode ser fechada pressionando as teclas ctrl + c no prompt de comando(cmd). Assim você encerra o programa e poderá fechar o prompt de forma segura, para que não fique funcionando em segundo plano podendo acarretar em travamentos necessitando assim que reinicie o computador.
